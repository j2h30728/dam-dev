dam-witter : 담위터

dam-witter 프로젝트에 관련된 내용입니다.

📄️ 소개

친구들과 함께 소소하게 일상을 나누는 커뮤니티

📄️ 이미지 업로드 로직 변경

- 문제: Cloudflare 이미지 등록 시간으로 인해 게시글 등록 시간이 지연되는 문제 발생.

📄️ 불필요한 이미지 등록

- 문제: 사용자가 연속적으로 이미지를 변경할 때 불필요한 이미지가 Cloudflare에 등록되는 문제가 발생.

📄️ 데이터베이스 리전

- 문제: 프로젝트 초기 단계에서 API 응답 시간이 지나치게 느려서 성능 저하 문제를 겪었습니다. 이 문제는 데이터베이스 리전이 미국으로 설정되어 있었기 때문에 발생했습니다.

📄️ '좋아요' 버튼 중복 요청 문제

- 문제: 사용자가 '좋아요' 버튼을 연속적으로 클릭할 때 중복 API 요청이 발생.
